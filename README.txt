WHAT THIS IS
────────────
This is the complete NOCTYRA360™ system in one folder.
V13 dashboard + production CDR engine + API + reports.
Everything integrated. One command to run.

HOW TO START (any computer with Python 3.9+)
─────────────────────────────────────────────
  bash run.sh

Then open your browser: http://localhost:8000/

That's it. The government opens the same V13 dashboard they saw in
the demo. The difference: now it can process the REAL CDR files
from the operators — any size, any format.

WHAT IT DOES
────────────
1. Upload a real CDR file from any operator (Telma, Orange MDG,
   Vodacom, Movitel, Airtel, etc.)
2. Server auto-detects the format (Huawei, Ericsson, Nokia, ZTE...)
3. Processes every row — 1 million to 1 billion records
4. Calculates EFR (Estimated Fiscal Recovery)
5. Detects SIM Box fraud, IMEI clones, MoMo money laundering
6. Generates SHA-256 certified PDF + Excel report
7. Dashboard displays results identically to the demo

FIRST TIME WITH A NEW OPERATOR
──────────────────────────────
- System auto-detects the CDR format
- Maps columns automatically (85-90% of cases)
- If a column is unrecognised: maps once, saved forever
- Next month same operator: fully automatic

FILE SIZE
─────────
- Demo files (MB): runs in browser, instant
- Real country files (GB): processed on server, chunked
- No limit — handles 50M+ rows per operator

REPORTS GENERATED
─────────────────
Every processing run produces:
  - JSON finding (machine-readable, API-accessible)
  - PDF report (SHA-256 certified, court-admissible)
  - Excel report (for DGI/AT/ARTEC use)

Reports saved in: reports_out/ folder

Add this file to your backup.
